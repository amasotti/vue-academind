FriendList

We have displayed here some names from an array. One way to do this is to populate an array, let's say "data" and grasp the data using a for-loop. This is fine but we would encounter a problem when trying to activate buttons or specific functionalities for just one of the items (like the show-details button here ). The much more vue-like way to achieve this is through Components : reusable pieces of code, actually mini Vue-app depending on the main app.
