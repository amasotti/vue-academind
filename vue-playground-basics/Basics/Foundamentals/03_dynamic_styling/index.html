The data() object was used here to keep track of the divs selected.

Binding a class or style element is different than binding other HTML attributes: here the syntax with short-hands:

        <div id="id" :style="{ 'borderColor' : boxA ? 'red' : '#ccc'}" ></div>

This method presupposes however the use of dynmamic style inline, which is not typically done in modern applications. Another option is to add a class to the css with the desired styling after the event (clickling for instance) and dynamically add or remove this class with Vue. 

          <div :class="{demo : true, activeDiv : boxC }" @click="boxSelected('C')"> 
          </div >
