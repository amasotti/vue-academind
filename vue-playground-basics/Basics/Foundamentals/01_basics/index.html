The element above were rendered in different ways with the aim to illustrate how things can be done in Vue:

  • Counters Both simply render the Counter property of data(), but the first one has one additional directive: v-once which freezes the counter to the value rendered when Vue is loaded
  • Hello World is rendered through the v-hmtl directive of Vue. It's also a variable in Data but contains HTML code. Withouth that directive Vue would automatically block HTML rendering and sanitize the code to prevent XSS-Scripting
  • Master Vue js is a single String variable in data
  • The successive 3 points are elements of an array rendered with a v-for directive
  • Goal for more/less is rendered through a method declared inside the methods in the Vue app
  • External func is declared outside the main Vuejs app but rendered through a method invoking it inside the app
